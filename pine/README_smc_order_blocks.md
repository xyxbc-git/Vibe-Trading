# SMC 订单块指标（smc_order_blocks.pine）使用说明

TradingView Pine Script v5 指标：自动识别并框选 SMC（Smart Money Concepts）订单块。

## 导入 TradingView

1. 打开 [TradingView](https://www.tradingview.com/chart/) 图表页面。
2. 点击底部面板的 **Pine 编辑器**（Pine Editor）。
3. 全选删除编辑器内的默认代码，将 `smc_order_blocks.pine` 的全部内容粘贴进去。
4. 点击 **保存**（起名如 `SMC OB`），再点击 **添加到图表**（Add to chart）。
5. 指标会以半透明色块出现在主图 K 线下方；在图表左上角指标名称处点齿轮可调参数。

## 识别逻辑

### 多头看涨订单块（默认红色）
1. **前置走势**：突破前一段为连续下跌或横盘震荡（净涨跌幅受"前置走势容差"约束）。
2. **触发**：一根实体力度达标的大阳线，收盘价向上突破之前 N 根 K 线的最高价。
3. **确认**：突破后连续 M 根 K 线不再回落跌破该阳线开盘价（M 可调，回破判定可选收盘价或影线极值）。
4. **标注**：自动向前回溯，框选突破阳线之前整段下跌/盘整 K 线区间，并在突破 K 线位置画竖虚线标记。

### 空头看跌订单块（默认青蓝色）
逻辑完全镜像：连续上涨/横盘 → 大阴线向下跌破之前 N 根 K 线最低价 → 破位后连续 M 根 K 线不再反弹站上该阴线开盘价 → 框选破位阴线之前整段上涨/盘整区间。

### 失效规则
- 收盘价**完全穿越**区块（多头块被向下击穿 / 空头块被向上击穿）→ 立即失效。
- 收盘价深入区块内部（越过区块中线）并**连续停留** N 根 K 线 → 失效。
- 失效后区块自动淡化不再高亮（也可设置为直接删除）。

### 防重叠与数量保护
- 同方向新区块与现存有效区块价格重叠超过阈值（默认 50%）时不重复生成。
- 同方向两次突破间隔小于"最小间隔"（默认 5 根）时不重复生成。
- 最多保留 60 个区块（可调），超出自动删除最旧的；脚本声明 `max_boxes_count=500`，不会触顶报错。

### 不重绘保证
所有判定均在 K 线**收盘确认后**执行（`barstate.isconfirmed`），历史回看与实时盘中行为一致；绘图锚定 `bar_index`，缩放、平移、切换周期均不错位。1 分钟到周线全周期可用。

## 使用建议

- **顺势交易**：价格回踩红色多头块上沿可关注做多机会；反弹至青色空头块下沿可关注做空机会。
- **周期适配**：小周期（1m/5m）建议调低"突破区间检测长度"至 6-8、确认 K 线数 2；大周期（4H/日线）可保持默认或调高。
- **信号过强/过弱**：区块太多→调高实体阈值（ATR 倍数 1.5+）或改用"两者皆需"模式；区块太少→调低至 0.6-0.8 或改用"实体占比"模式。
