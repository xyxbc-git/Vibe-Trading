# 贾维斯桌面交易终端 — 开发计划

> 版本：v1.0 | 日期：2026-06-24 | 状态：全部完成
>
> 目标：构建一个专业级 macOS 桌面交易终端，双击即用，无需终端命令。

---

## 0. 目标

把贾维斯的所有能力（K线图表、交易执行、策略进化、市场情报、AI问答）封装成一个
**类似币安 Pro 的深色主题桌面应用**，让用户：

1. 双击 `.app` 即可使用，无需打开终端
2. 一眼看到持仓、收益、市场情绪
3. 一键启动策略进化、自动交易
4. 随时向 AI 提问市场分析

---

## 1. 技术栈

| 层级 | 技术 | 说明 |
| --- | --- | --- |
| 桌面框架 | Electron 33+ | 成熟稳定，币安/OKX 桌面端同款 |
| 前端框架 | React 19 + TypeScript | 组件化开发 |
| 构建工具 | Vite 6 | 极速 HMR 开发体验 |
| 样式方案 | TailwindCSS 4 | 暗色主题快速搭建 |
| K线图表 | TradingView Lightweight Charts v4 | 专业级交易图表 |
| 数据图表 | Recharts | 收益曲线、饼图、雷达图 |
| 动画 | Framer Motion | 流畅过渡动效 |
| 图标 | Lucide React | 一致的图标风格 |
| 后端 | Python FastAPI（已有 :7899） | 26+ API 端点复用 |
| 打包 | electron-builder | macOS .app 打包 |

| `/api/evolve/start` | POST | 启动进化（参数：rounds） |
